# HWAX Agent Server 기능·마일스톤 요약 (비개발자용)

> **HWAX Agent Server** = HWAX 포털 MCP 채팅의 "두뇌". 실제 AI 모델 호출과 업무 도구 실행을 담당하는 별도 서버.
> 포털은 얇은 통로(인증·중계)일 뿐, 진짜 일은 여기서 처리.

작성 기준: HWAX 포털 통합 대상 시스템 정리 (비개발자용 개조식)

---

## 주요 기능 6

### 1. AI 두뇌 연결
- 실제 답을 만드는 AI 모델에 접속
- 개발용(작은 두뇌)·운영용(큰 두뇌) 전환은 주소 한 줄 교체
- 모델 교체에 코드 수정 불필요

### 2. 에이전트 진행
- 단순 응답이 아니라 "도구가 필요한지" 스스로 판단
- 도구 실행 후 그 결과를 근거로 답변
- "지어내지 말 것" 지침 내장

### 3. 업무 도구 연동
- 외부 업무 프로그램을 AI 에 장착
- 현재 보고서 작성·VOC(고객의 소리) 분석 등 26개 기능
- 도구 추가는 설정만으로 가능

### 4. 설정·보안 관리
- 연결 도구 목록·접속 열쇠를 코드와 분리 보관
- 비밀정보 외부 노출 차단
- 환경별 설정 분리

### 5. 실시간 응답
- 답을 글자가 흐르듯 즉시 출력
- "분석 중"·"도구 사용 중" 진행 상황 표시
- 포털 화면 규격에 맞춰 전달

### 6. 무중단·안전
- 도구 고장 시에도 가능한 선에서 답변
- 오류 시 사용자엔 안전한 메시지만 노출
- 내부 정보 차단

---

## 개발 마일스톤 5단계

### M1. 골격
- AI 답을 그대로 화면에 전달
- 기초 챗봇 단계 (도구 없음)

### M2. 에이전트화 ★분기점
- 도구를 직접 쓰는 비서로 전환
- 도구 실제 사용까지 검증 완료

### M3. 하드닝
- 오류 시 내부 정보 감춤
- 보안·신뢰 안전장치

### M4. 실전 툴 투입
- 연습용 도구 → 실제 업무 도구(보고서·VOC)
- 데모 졸업, 제품 단계 진입

### M5. 라이브 (현재)
- 26개 도구 정상 가동 중
- 운영 전환은 주소 교체만 남음

---

> M1~M3 은 토대(전달·에이전트화·안전), M4~M5 는 실제 업무 도구 장착·정상 가동 단계.
