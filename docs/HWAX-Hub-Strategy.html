02 각 팀이 해야 할 일 — 4가지 유형 팀 상황에 맞게 선택 · 복수 참여 가능
🔌
유형 1 — 자체 서비스 연동
이미 자체 서비스가 있는 팀
팀 서비스를 계속 자체 운영하되, 허브와 데이터를 주고받을 수 있도록 규격 협의
정해진 API 규격(데이터 전송 방식)에 맞춰 연동하면 포탈에서 통합 접근 가능
연동 담당자 1명 지정 필요 — 규격 협의 및 지속 관리 역할
💬 허브팀과 연동 규격 협의 → 담당자 지정 후 시작
⚙️
유형 2 — 서비스 운영 위탁
개발은 하지만 운영은 맡기고 싶은 팀
개발·테스트는 자체적으로 진행 (기존과 동일)
완성된 소스코드를 정해진 방식으로 GitHub에 올리면 허브가 자동으로 서비스 개시
배포·서버 관리·장애 대응을 허브가 대행 → 팀은 개발에만 집중
지원 가능 언어: Python, Java, Node.js, C++, Go, R 등 15종 이상
📋 소스코드 등록 규격 안내 후 GitHub 업로드만 하면 자동 운영
🤖
유형 3 — AI Agent / 데이터 활용
AI 검색·자동화를 활용하고 싶은 팀
그룹 데이터 담당자 지정 — AX Data Hub 데이터·Agent 업로드 관리
업무 데이터·문서를 정해진 형식으로 업로드하면 AI 검색·질의응답 즉시 활용 가능
반복 업무를 처리하는 AI Agent 등록 및 운용 (담당자가 직접 관리)
ChatGPT / Claude 오픈 후 — 데이터 업로드 방법 LLM 활용 가이드 공유 예정
🔑 담당자 지정 → 업로드 규격 안내 → AI 서비스 즉시 사용
📘
유형 4 — 업무 백서 등록
팀 핵심 업무 지식을 AI로 활용하고 싶은 팀
팀의 주요 업무 지식·노하우·매뉴얼을 백서로 정리하여 MX White Paper에 등록
등록된 백서는 포탈 전체에서 검색·참조 가능 → 조직 내 지식 공유 가속
AX Data Hub와 연계 시 백서 기반 AI 질의응답 서비스 자동 구성
ChatGPT / Claude 오픈 후 — LLM을 활용한 백서 작성 방법 가이드 공유 예정
📝 백서 작성 가이드 제공 → 업로드 → 즉시 AI 검색 적용
