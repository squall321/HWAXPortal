# AI Data Hub 기능 요약 (비개발자용)

> 사업부 데이터(문서·측정·시뮬레이션·CAD)를 통합한 데이터 카탈로그 + AI 검색·분석 서버.
> HWAX Portal 의 `/ai-data-hub/` 타일로 연결됨.

---

## 주요 기능 6가지

### 1. 데이터 적재·정리
- 문서·측정·시뮬레이션·도면을 한곳에 통합
- 워드·엑셀·PDF·PPT 자동 변환 → 검색 가능한 형태로 저장
- 수동 타이핑 불필요

### 2. 똑똑한 검색
- 의미 기반 검색 (단어가 정확히 안 맞아도 비슷한 자료 탐색)
- 단어·의미·태그 검색 혼합
- 관련도순 정렬

### 3. 분야별 AI 상담원
- 분야별 전문 상담원 설정 (VOC·재료시험 등)
- 각 상담원은 자기 분야 자료만 근거로 답변
- 질문 시 최적 상담원 자동 추천

### 4. 외부 시스템 자동 연동
- 사내 다른 시스템(VOC 수집·사내 위키)에서 30분마다 자동 수집
- 사람 개입 없이 최신 상태 유지
- 중복 제거 후 한 건만 저장

### 5. Claude 채팅으로 다루기
- 채팅으로 자료 등록·검색·정리
- 표 붙이면 자동 저장 + 부족분만 되묻기
- 비슷한 자료 찾아 분류 제안

### 6. 무인 운영
- 서버 꺼져도 자동 재가동
- 매일 자동 백업
- 문제 발생 시 스스로 복구

---

## 개발 마일스톤 6단계

### M1. 자료 창고
- 흩어진 자료를 통합 검색 창고로 구축
- 워드·엑셀·PDF 자동 변환 저장

### M2. AI 상담원
- 분야별 상담원이 데이터를 근거로 답변
- 의미 기반 추천·검색

### M3. 자동연동·무인운영
- 외부 시스템 자료 자동 수집
- 자동 복구·검색 속도 개선

### M4. 배포·포털 연결
- 어느 서버에도 설치 자동화
- 매일 백업·사내 포털 통합

### M5. Claude 직접 활용
- 채팅으로 자료 등록·정의
- 표 자동 저장·부족분 되묻기

### M6. 분석하는 AI
- AI 가 표를 직접 계산 (평균·최대 등)
- 그래프 도구 자동 선택

---

> M1~M4 는 토대(데이터 창고·검색·연동·배포), M5~M6 은 "채팅만으로 자료를 등록하고 분석까지 시키는" 단계.
