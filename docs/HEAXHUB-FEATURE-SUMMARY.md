# HEAXHub 기능·마일스톤 요약

> **HEAXHub** = 사내 개발자가 만든 프로그램을 한곳에 모아, 자동으로 포장해서 누구나 웹에서 찾아 쓰게 해주는 사내 자동화 포털.
> 한마디로 "사내 앱스토어 + 자동 배포 공장".

작성 기준: HWAX 포털 통합 대상 시스템 정리 (비개발자용 개조식)

---

## 기능 6

| # | 기능 | 핵심 내용 |
|---|---|---|
| ① | 프로그램 접수·심사 | 개발자 제출 → AI 자동 분석·설정 제안 → 담당자 검토·승인 → 공개 (단계별 결재형) |
| ② | 자동 포장·공개 공장 | 22종 기술 프로그램 자동 패키지화 → 사내 웹 한곳에 주소별 게시 → 끊기면 자동 복구 |
| ③ | 실행 엔진 | 일반 서버·슈퍼컴퓨터·직원 PC 구분 실행 + GPU·유료 라이선스 자동 배분 |
| ④ | 보안·출입 관리 | 로그인·권한 통제 + 비밀 열쇠 암호화 금고 + 행위 기록(감사) + 회사 통합 로그인 연동 |
| ⑤ | 직원 PC 연동·외부 연결 | Windows PC 도우미 설치·연결 + 설치파일 자동 배포·자동 업데이트 + GitHub 연동 |
| ⑥ | 스스로 설치·복구 배포 | 사전 설치 거의 불필요 자가 설치 + 폐쇄망은 구글 드라이브 우회 + 다운 시 자동 복구 |
