설명 우리가 안내할 것
Report Archive·백서 사이트는 위젯 기반으로 복잡한 데이터 접근성이 확보돼, 사내 위키와 달리 LLM의 데이터 인식이 수월함을 설명.
PPT를 모두 넣으면 내용이 잘 정리되는 점 — Report Archive 적재의 실효성을 안내.
요청 각 팀에 요청
Claude·Gemini·ChatGPT로 여러 주제를 선정해 PPT를 다수 생성·적재하고, 전문 인력이 읽으며 LLM으로 내용 augment 진행 요청.
각 팀 raw data를 MCP로 허브에 올리기 위해, 각자 올릴 수 있는 데이터 예시 요청.
제안 제안할 것
데이터 취합 후 분야별 백서화는 시스템으로 자동화 — 각 팀은 검수만 진행하도록.
각 팀 포털의 통합 진입점 논의 + MCP Gateway로 각 팀 Agent 통합 관리 제안.
