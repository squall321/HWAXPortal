# MX White Paper 기능 요약 (비개발자용)

> 축적된 업무 지식과 노하우를 AI 가 정리·작성하는 사내 백서 플랫폼. 흩어진 경험을
> 검색 가능한 지식 자산으로 전환해 팀의 집단 지능을 끌어올린다.
> HWAX Portal 의 `/mx-white-paper/` 타일로 연결됨.

---

## 주요 기능 6가지

| # | 카테고리 | 핵심 내용 |
|---|---|---|
| 1 | 문서 작성 기본 틀 & 콘텐츠 블록 | 백서 편집기, 38종 콘텐츠 조각(표·차트·강조박스 등), 문서 양식 자동 통일 |
| 2 | 파일 가져오기 / 내보내기 | 워드·PPT·엑셀·PDF 올리면 백서로 자동 변환, 백서를 워드·PDF 로 재내보내기, 기존 자료 그대로 이전 |
| 3 | 데이터 분석 위젯 | 엑셀 피벗급 요약·교차분석, 부서·기간 조건 필터, 숫자 클릭 시 근거 원본 확인 |
| 4 | 디자인 & 사용 편의 | 차트 시각화, 다크모드, 발표용 슬라이드 보기, 모바일 대응, 한/영 전환, 접근성 |
| 5 | 지식 연결 & 검색 | 용어·문서 연결 지도, 통합 검색, 작업기록·지식 AI 재활용 |
| 6 | 안정 운영 & AI 연동 | 자동 백업·복구, 회사 포탈 편입, AI(Claude) 의 백서 직접 읽기·쓰기·그림 삽입 |

---

## 개발 마일스톤 6단계

| # | 시기 | 마일스톤 | 핵심 |
|---|---|---|---|
| M1 | 5월 2주차 | 첫 동작 버전 | 작성·저장·내보내기 기본기 완성 |
| M2 | 5월 3주차 | 콘텐츠 블록 + 파일 가져오기 | 표·차트 조각 구비, 워드→백서 변환 |
| M3 | 5월 4~5주차 | 보여주는 힘 강화 | 다크모드·발표 보기·모바일, 다국어, 용어·문서 연결 지도 |
| M4 | 5월 5주~6월 1주차 | 다루는 힘 강화 | 문서 내 피벗·조건 필터·근거 추적 |
| M5 | 6월 2주차 | 포탈 통합 + 운영 안정 | 메인 포탈 편입(통합 로그인), 자동 백업·복구 |
| M6 | 6월 3~4주차 | AI 연동 + 이식성 | AI 백서 작성·수정·그림 삽입, 서버 이전 무관 동작, 배포 자동화 |

> 발전 흐름 — 기본기(M1·M2) → 잘 보여주기(M3) → 잘 분석하기(M4) → 회사에 안정적으로 올리기(M5) → AI 가 대신 일하게 하기(M6).
